# 🌸 Women’s Health & Wellness Website

A modern, responsive, and visually appealing web application designed to support women’s physical health, mental wellness, and overall quality of life. This project focuses on creating a calming, empowering, and interactive digital experience for women of all ages.

---

## ✨ Features

### 🏠 Hero Section

* Inspiring headline and empowering message
* Call-to-action button: **“Start Your Wellness Journey”**
* Animated gradient background

### 💪 Physical Health

* Fitness, nutrition, and sleep tips
* Interactive cards with hover effects
* Icon-based UI for easy understanding

### 🧠 Mental Wellness

* Stress management & mindfulness tips
* Expandable/collapsible sections
* Clean and calming UI

### 🌺 Women-Specific Health

* Menstrual health, pregnancy, PCOS awareness
* Accordion / tab-based layout

### 📊 Daily Wellness Tracker

Track your daily habits:

* 💧 Water intake
* 😊 Mood
* 🏃 Exercise

> Built using JavaScript for dynamic interaction

### 💬 Testimonials / Quotes

* Smooth carousel/slider
* Inspirational wellness quotes

### 📚 Blog / Resources

* Informative articles
* Card-based layout with hover effects

### 📩 Footer

* Contact details
* Social media icons
* Newsletter signup form

---
